# Monday.com Power

## Overview

This power provides integration with Monday.com for managing boards, items, and workflows.

## Available MCP Servers

### monday-api-mcp

The Monday API MCP server provides tools for:

- Creating and managing boards
- Adding and updating items
- Managing groups and columns
- Searching across workspaces

## Configuration

Requires a Monday.com API token. Get yours at https://monday.com/developers/apps
